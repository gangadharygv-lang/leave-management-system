# leave-management-system
# Enterprise Leave Management System with Data Analytics

## 📌 Project Description
This is a web-based application developed using Java (JSP & Servlets) to manage employee leave requests efficiently.

The system automates leave application, approval, and provides analytical insights.

---

## 🚀 Features
- User Login Authentication
- Apply Leave
- View Leave Status
- Manager Approval/Rejection
- Dashboard Analytics (Total, Approved, Rejected Leaves)

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS
- Backend: Java (Servlets, JSP)
- Database: MySQL
- Server: Apache Tomcat

---

## 📊 Analytics
- Total Leaves Count
- Approved Leaves
- Rejected Leaves

---

## 📁 Modules
- Login Module
- Leave Request Module
- Leave Approval Module
- Report & Analytics Module

---

## 🎯 Outcome
- Reduces manual work
- Improves transparency
- Provides insights for decision-making
